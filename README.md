# Lilmao Gaming - Personal Creator Website

A personal website for gaming content creators featuring social media links and KHQR donation integration.

## Features

- 📱 Responsive design for all devices
- 🔗 Social media links (YouTube, Facebook, TikTok)
- 💰 KHQR donation integration for fan support
- 🎨 Modern and clean UI
- ⚡ Fast and lightweight

## Project Structure

```
Lilmaogaming/
├── index.html          # Main landing page
├── css/
│   └── style.css       # Styling
├── js/
│   └── script.js       # JavaScript functionality
├── assets/
│   ├── images/         # Images and media
│   └── Tmi/            # Temp/misc assets
└── README.md           # This file
```

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- KHQR API Integration

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize with your social media links and donation details
